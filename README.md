## vrchat-osc-motd
Customizable MOTD (in chatbox) system with plugins support for VRChat.

### Features
- Lightweight
- Zero config
- Plugins system

### Default plugins
- AFK Detecting
- MOTD
- PC Stats
- Spotify Song <i>(linux only yet)</i>
